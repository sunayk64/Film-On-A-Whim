# Film-On-A-Whim
watch a film on a whim. 


Step 1: 
 Pull the repo to your own computer, and create a folder called "data" in the main home folder. Then, download 'title.basics.tsv' and 'title.ratings.tsv' from the link: 
 https://datasets.imdbws.com/
 after downloading these unzip and put the inner two files into the data folder.
